Computer Setup

Why so many un-necessary programs?

When I first setup this computer I was pretty new to the Windows 10 interface, so I will say that I am no expert here, but I do know Windows 10 still has bugs which are in need of fixing. What I did bring to the table was a basic understanding of how computer manufacturers are able to provide decent machines, while still keeping their customer's costs down. Sadly this practice can lessen the computer's ability to perform the tasks that we ask of it.
The producers of many computers in this day and age will sell a computer with all sorts of un-necessary content pre-loaded onto it because they are effectively paid to include this software. This content is 'sponsored' by the software company. A good example of this is the McAfee antivirus software which came with a one year free trial standard with this computer. Little do many users of this software realize, that this free trial is actually quite similar protection of that which can be obtained through various antivirus programs which are actually free forever. When this year trial is up, this program is somewhat difficult to remove and can act as a sort of spyware in itself. This is certainly effective 'advertising'.

As a matter of fact, when I initially removed the installation of the Mcaffee program from this computer and attempted to install a different antivirus program, this 'spyware' type of functionallity that I had mentioned kicked in. Upon making my way to one of Mcaffee's competitors web sites, I was automatically re-routed back to Mcaffee's own website! This caught me off gaurd as it was the first time I had experienced this first hand. Talk about manipulation!

My Initial Setup Choices

My steps in preparing this computer, and any for that matter, are relatively simple.
  • Remove un-necessary junk that will take up resources better utilized elsewhere
  • Implement a labyrinth of programs for protecting and maintaining the system through a plethora of ways it might be used
  • Complete these tasks using only software that is both legal and actually free. No 'free' trials here

    Note: The following programs all have free and paid versions. For purposes described here, we will only be making use of the free versions of this software.

Evacuate the free loaders! - In removing the un-necessary programs, precious resources will be freed up for more preferable tasks. I simply looked through the installed program list (Right click on start menu -> settings -> apps & features) and removed anything that I knew I wouldn't be using, or was not essential for the computer to run.

As it has been a while since I have done this, I do not have a lot of specific examples, but for one, I knew I was not interested in One Drive, so I uninstalled it. Remember, these can be re-installed if later you decide you want to make use of a given program.

When it comes to software that will maintain a computer's performance and security, there are too many options available to count. The software shown here is just my preference as to what has worked for in my situation for many years. So without any further blabbering, here is what I like to use.

First off, we will defantly be needing some form of virus protection. My favorite used to be AVG; however, in the past couple of years has changed to a program called Avast. Oddly enough, I recently discovered via Wikipedia, that Avast bought out AVG in 2016 for some $1.3 Billion! Avast contains some very useful features that other programs simply do not offer. One such feature includes the ability to scan for viruses and malware before your operating system has a chance to boot. This can grant the antivirus program the ability to look in areas that would otherwise be unaccessible as the operating system would be using these files to run. This is a prime location for various threats to remain undetected.

To visit the Avast website click HERE.

Next it is recommended that a computer has some sort of program which will locate and delete files that simply build up from every-day use and do not serve much purpose aside from their initial purpose. My favorite program for such occasions is called CCleaner. The full name for this program is actually 'Crap Cleaner', but is not openly advertised as such, likely for the case of remaining professional. Including more background than necessary as I do, it has recently come to my attention that the makers of CCleaner, Piriform, has also been bought out by the Avast company back in 2017. This information presented by wikipedia. CCleaner offers many useful tools other than simply cleaning up temporary system and internet files, you can also modify what programs will start with Windows, clean out your system's registry files, locate duplicate files for additional hard drive clean up, wipe drive space, and more.
